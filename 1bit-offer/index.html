Веду задачу от данных до работающего агента: беру узкий кусок, за дни довожу до живого прототипа, обкатываю на реальных данных, дальше — тесты, деплой, поддержка. Не «ещё один скрипт», а система, которую можно развивать.

A

AI-агенты

Не чат-болталка: агент с инструментами — сам ищет, отвечает и выполняет действия. Необратимое — только через подтверждение человека.

M

Слой памяти / RAG

Семантический поиск по данным: находит по смыслу, отвечает со ссылкой на источник. Локальные эмбеддинги — без утечки данных наружу.

↔

Интеграции

Связываю платформы в один контур: календарь, трекеры, мессенджеры, внешние API — и данные начинают работать вместе.

↻

Автоматизация

Рутину — на фоновых агентов: делают сами, уведомляют о результате, отчитываются о статусе.

Стек: Python, LLM (Anthropic/локальные), RAG, MCP-инструменты, интеграции по API, деплой и мониторинг.

Как подключиться

Начнём с одного департамента

Предлагаю проверить теорию на реальном, но небольшом участке — один департамент, один сценарий. Так виден результат без риска для остальных процессов, и дальше масштабируем только то, что сработало.

01

Один участок и сценарий → работающее демо за пару недель на ваших / тестовых данных. Проверка теории без риска.
02
