01

Не приложение

POS — это не конкретный продукт или сервис. Это среда, которую ты выстраиваешь под себя.

02

Не модель

В центре — не AI. В центре ты: твой стиль, цели, ограничения, ритм работы.

03

Персонализированные агенты

Микросервисы, промпты, пайплайны — всё соединено через единый личный контекст.

04

Живая инфраструктура

Не теоретическая концепция, а рабочая система, которая знает о тебе всё, что нужно.

СЛОЙ 05
▣

Стек инструментов

Obsidian, Linear, Calendar, Telegram, Claude Code — узлы личной среды.

Слой 02 — Агенты

Не боты. Твои операторы.

RESEARCHER

Исследователь

Копает глубоко, синтезирует, фильтрует через твои приоритеты и контекст.

EDITOR

Редактор

Знает твой голос, стиль и аудиторию. Не исправляет — доводит до идеала.

COORDINATOR

Координатор

Управляет потоком задач, делегирует другим агентам, следит за контекстом.

ANALYST

Аналитик

Превращает данные и встречи в структурированные инсайты под твои решения.

PLANNER

Планировщик

Строит roadmaps и недельные планы с учётом твоего ритма и ограничений.

COMMS

Коммуникации

Пишет письма, посты, брифы — в твоём тоне, с твоим намерением.

Как это работает изнутри

Skills, MCP
и CLAUDE.MD

CLAUDE.MD / AGENTS.MD

Главный файл конфигурации AI-агента — глобальный или проектный. Задаёт контекст, правила и поведение агента с первой секунды.

MCP
📅 Calendar ⚡ Linear ✈️ Telegram 📁 Files 💻 Code

Как «USB-порты» для AI-агента. Через MCP агент подключается к Calendar, Linear, Telegram и любому другому инструменту.

Один стандарт — не костыли для каждого инструмента
SKILLS

Markdown-файл с пошаговой инструкцией для агента. Как стандартная операционная процедура для сотрудника — не нужно каждый раз объяснять с нуля, как делать задачу.

Переиспользуемые системные промпты с файлами — только более структурированные и с триггерным механизмом
Ресурсы подгружаются по необходимости
Триггер активирует нужный скил автоматически
Один скил — одна повторяемая практика
